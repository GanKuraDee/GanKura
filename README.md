# GanKura
GanKura is a Hypixel Skyblock Mod focused on End Stone Protector.\
Inspired by GolemUtils https://chattriggers.com/modules/v/GolemUtils

Type /gankura to open the settings menu.

# Features
<details><summary>End Stone Protector</summary><ul><li><strong>Golem Stage Display</strong></li><ul><li>Displays the golem's stage on the HUD.</li></ul></ul><ul><li><strong>Golem Location Display</strong></li><ul><li>Displays the golem's spawn location on the HUD.</li></ul></ul><ul><li><strong>Golem Spawning Countdown</strong></li><ul><li>Displays 20s countdown on the HUD until the golem spawns. (Considering the server ticks)</li></ul></ul><ul><li><strong>Golem HP Display</strong></li><ul><li>Displays the golem's HP on the HUD.</li></ul></ul><ul><li><strong>Stage 4 Duration Display</strong></li><ul><li>Displays the time it took for the golem to reach Stage 5 on the HUD and chat.</li></ul></ul><ul><li><strong>DPS Calculator</strong></li><ul><li>Shows your DPS you have dealt to the golem in the chat. (Considering the server ticks)</li></ul></ul><ul><li><strong>Loot Quality Calculator</strong></li><ul><li>Shows your golem's loot quality in the chat.</li></ul></ul><ul><li><strong>Rare Drop Notification</strong></li><ul><li>Display rare drops from golems in title and chat.</li></ul></ul><ul><li><strong>Golem Loot Tracker</strong></li><ul><li>Displays the HUD with rare drops from golems so far.</li></ul></ul><ul><li><strong>Golem World Location Display</strong></li><ul><li>Displays the golem's spawn location on the world.</li></ul></ul></details>
